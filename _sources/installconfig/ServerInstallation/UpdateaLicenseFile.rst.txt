How to Update a License File
=====================================================================

This article is intended for customers who need to update an expiring
license file. It provides the steps for updating your expiring license
file.

Prerequisites
--------------

Requires Alation 4.10.0 or higher.

Update the License File
-----------------------

To update a license file:

#. Visit the `Customer Portal <https://customerportal.alationdata.com>`_ to get the latest copy of your license file.
#. Select an instance for which you want a new license file.
#. Log in to your instance of Alation.
#. Click the **Info** icon on the right of the navigation bar then click **About this Instance**:

    .. image:: ../../_static/aboutThisInstance.png
        :width: 221.50px
        :height: 130.93px
        :align: center
        :alt: image

#. Scroll to the bottom of the page and click the **Upload License File...** button:

    .. image:: ../../_static/uploadLicense.png
        :width: 500px
        :align: center

#. Select your license file using the file chooser window and upload to the instance.
