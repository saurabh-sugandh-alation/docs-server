Restore Alation Analytics V2
==============================

.. include:: /shared/ProductLabels/CustomerManaged_Label.rst

The Alation Analytics database is not backed up by the Alation backup process. If you are restoring an Alation instance from a backup, the Alation Analytics database requires additional steps to be set up after the restoration.

For more information, see :doc:`/installconfig/AlationAnalyticsV2/BackupAndRestoreAlationAnalyticsV2`.
