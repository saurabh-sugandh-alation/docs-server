How to Create a Project in GCP
------------------------------

Reference: `Creating and Managing Projects <https://cloud.google.com/resource-manager/docs/creating-managing-projects>`_.

#. Open the `Google Cloud Platform <https://console.cloud.google.com/>`_  (GCP) console.
#. Click **IAM & admin**. 
#. Click **Manage resources**.
#. On the **Manage resources** page, click **Create Project**.
#. Enter the **Project Name**.
#. Click **Browse** to choose the Location.
#. Click **Create**. The project gets created.

You can enable billing for the project. For more information on how to enable billing, refer to this topic in `GBQ documentation <https://cloud.google.com/billing/docs/how-to/change-project>`_.

BigQuery is automatically enabled in new projects. This helps you to access the APIs.

.. note::

   The Project you use in Alation should be billing-enabled.
