Add a File System
====================

To add a file system,

1. Log in as a *Server Admin*. Click **Sources** on the main toolbar then click the **Add**Â icon and select **File System**.

  .. image:: ../../_static/file_image23.png
        :width: 624px
        :align: center
        :height: 88px
        :alt: image

2. This brings up the **Add a File System** wizard. Select the appropriate **File System Type**.
3. Enter the **Title**.
4. Click **Add File System**.

      .. image:: ../../_static/file_image7.png
          :width: 300px
          :align: center
          :alt: image

5. The **Settings** page of the selected File System will open. Enter the file system-specific values for the connection, authentication, extraction settings, and Privacy settings. File System Privacy settings are available from version V R7 (5.12.3).
