Labs
========

*Applies to release 2020.4 and earlier versions*

.. note::

    In versions 2021.1 and newer the **Labs** page is :doc:`Feature Configuration <../../admins/AdminSettings/FeatureConfiguration>`


Labs is a section on the Admin Settings page where you can enable and disable some of the features. It is available to users with the Server Admin role.

To open Labs,

1. Sign in to Alation as a *Server Admin* and click the **Admin Settings** icon on the upper right to open the Admin Settings page:

    .. image:: ../../_static/Labs_01.png
        :width: 5.5in
        :align: center

2. Under **Server Admin**, click **Labs** to open the Labs page:

    .. image:: ../../_static/Labs_02.png
        :width: 5.5in
        :align: center

3. In the list of features, find the functionality you want to enable or disable.

4. Click the corresponding toggle to enable or disable a feature. Note that some features cannot be disabled:

    .. image:: ../../_static/Labs_03.png
        :width: 5.5in
        :align: center

5. Changes on the Labs page require to be saved. Click **Save** on the upper right of the page to save the changes:

    .. image:: ../../_static/Labs_04.png
        :width: 5.5in
        :align: center

6. In the Save confirmation dialog that opens, confirm your action by clicking **Change Lab Actions**. The feature will be enabled (or disabled):

    .. image:: ../../_static/Labs_07.png
        :width: 5.5in
        :align: center

Some features cannot be disabled. This will be indicated in the feature description next to the feature toggle. For example:

.. image:: ../../_static/Labs_05.png
    :width: 5.5in
    :align: center

When you enable such a feature, the toggle will become inactive eliminating the ability to turn the feature off:

.. image:: ../../_static/Labs_06.png
    :width: 5.5in
    :align: center
