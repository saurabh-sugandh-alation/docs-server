Cloud Sandbox
=============

.. include:: /shared/SandboxDeprecated.rst

Cloud Sandbox emulates a backup copy of your Alation Cloud Service production instance for testing new functionality, data sources, authentication, and new configurations. It's made available for two weeks per quarter. There are two Cloud Sandbox plans:

-  Cloud Sandbox

-  Cloud Sandbox Max

.. _cloud-sandbox-1:

Cloud Sandbox
-------------

Cloud Sandbox will not emulate the hardware requirements of your production instance and is not recommended for performance testing.

Cloud Sandbox Max
-----------------

Cloud Sandbox Max emulates the hardware requirements of your production instance and therefore is recommended for performance testing.
