Create a New File System Source
====================================

.. include:: /shared/ProductLabels/CloudAndCustomerManaged_Label.rst

To create a new file system source:

1. Log in to the Alation instance.

2. On the top right, click **Apps** and then **Sources**.

3. On the **Sources** page that opens, click **Add** on top right, and from the list of source types select **File System**. The **Add a File System** screen will open.

4. From the **File System Type** dropdown, select the appropriate connector.

5. Provide a **Title** for your file system source and click the **Add File System** button. You will be navigated to the Settings page of your new file system source.
