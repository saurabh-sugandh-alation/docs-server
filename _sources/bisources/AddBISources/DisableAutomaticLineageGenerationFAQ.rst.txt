Disable Automatic Lineage Generation FAQ
==============================================

Will the previously generated automatic lineage be removed if automatic lineage generation is disabled?
-------------------------------------------------------------------------------------------------------------

No. The previously generated will not be removed if automatic lineage generation is disabled.

Will the lineage data be available for the BI source to which lineage was loaded via the API, if i rerun the extraction and disable automatic lineage?
------------------------------------------------------------------------------------------------------------------------------------------------------------

Yes. The lineage data will be available.

Will the API-lineage data be available for the BI source to which lineage was loaded via the API, if i rerun the extraction and enable automatic lineage?
-----------------------------------------------------------------------------------------------------------------------------------------------------------------

No. The API-lineage will be overridden by the extracted lineage data.
