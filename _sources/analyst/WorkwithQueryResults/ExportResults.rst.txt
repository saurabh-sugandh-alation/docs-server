Export Results
=====================================================================

.. include:: /shared/ProductLabels/CloudAndCustomerManaged_Label.rst

Alation supports standard export functionality to CSV and CSV file types. To export your full result set, use the **Export** button instead of the **Run** button in Compose. Clicking on CSV or TSV from the **Run** dropdown menu sets the default export file type. After you have set your default and you can click **Export**.

.. image:: /_static/Compose_ExportMenu.png

.. note::

   There is a 100-million row limit for exports.
