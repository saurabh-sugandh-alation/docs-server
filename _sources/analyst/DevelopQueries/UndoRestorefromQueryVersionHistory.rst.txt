Undoing a Restore from Query Version History
=====================================================================

.. include:: /shared/ProductLabels/CloudAndCustomerManaged_Label.rst

*Available from version V R5 (5.9.x)*

It is possible to roll back a restore. To undo a version restore,

Click **Undo** on the **This query is restoredâ€¦** banner that
appears on the query page after you restore an older version. You will
return to the most recent version of the query.
