Inserting an Excel Table into an Article
=====================================================================

.. include:: /shared/ProductLabels/CloudAndCustomerManaged_Label.rst

There are three ways to insert an Excel table into an Alation Article.

1. Copy and paste the Excel table into the article.

    .. image:: ../../_static/InsertExcelTable_Paste.png

2. Attach the Excel sheet to the article using the **Attachments** field.

    .. image:: ../../_static/InsertExcelTable_Attach.png

3. Insert the table as an image and reference where users can find the table (an Attachment, a link, etc.).

    .. image:: ../../_static/InsertExcelTable_Image.png
