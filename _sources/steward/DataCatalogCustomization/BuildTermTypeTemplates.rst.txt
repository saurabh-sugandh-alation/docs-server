Build Term Type Templates 
=====================================================================

.. include:: /shared/ProductLabels/CloudAndCustomerManaged_Label.rst

After you have :doc:`developed your desired custom fields <../../steward/DataCatalogCustomization/CreateArticlesandArticleTemplateCustomFields>`, you can
begin to create custom templates for Term Types, policies, and articles. 

Custom fields can be used for any of :doc:`data object pages <../../steward/DataCatalogCustomization/CatalogTemplates>`, Term Type templates, Policy templates, or article pages. This topic focuses on Term Types.

To work with Term Type templates:

1. Log in to Alation as a Catalog Admin or Server Admin.
2. If you are a Catalog Admin, mouse over the settings icon on the right of the main toolbar:
       
    .. image:: ../../_static/Screen_Shot_2018-10-16_at_2.17.18_PM.png

    In the menu that opens, click **Customize Catalog**. The **Customize Catalog** page will open:

    .. image:: ../../_static/Customize_catAdminPath.png

    .. note::

         If you log in as a Server Admin, go to **Admin Settings** > **Catalog Admin** section > **Customize Catalog** to open the **Customize Catalog** page.

3. Click the **Custom Templates** tab.
4. Scroll to **Term Types**.
5. You can now select a template to edit or create a new template.

    .. image:: ../../_static/CFF.png

After you save your template, you can use it to start creating Glossary Terms.

Add a Custom Template
------------------------

1. Log in to Alation as a Catalog Admin or Server Admin.
2. If you are a Catalog Admin, mouse over the settings icon on the right of the main toolbar:
       
    .. image:: ../../_static/Screen_Shot_2018-10-16_at_2.17.18_PM.png

    In the menu that opens, click **Customize Catalog**. The **Customize Catalog** page will open:

    .. image:: ../../_static/Customize_catAdminPath.png

    .. note::

         If you log in as a Server Admin, go to **Admin Settings** > **Catalog Admin** section > **Customize Catalog** to open the **Customize Catalog** page.

3. Click the **Custom Templates** tab.
4. Scroll to **Article Templates**.
5. Click the add icon:

    .. image:: ../../_static/Custom_Add2.png

    A Term Type template editor opens on the right:

    .. image:: ../../_static/Custom_TermType.png

6. In the editor, enter the **Template Title**.
7. Insert **Custom Fields** to be included in the template. Name, Description, Domains, and Tags fields are included automatically. You can insert multiple fields, and rearrange the order of inserted fields. 
8. Click the **Save** button in the lower right of the editor to save your template:

    .. image:: ../../_static/Custom_SaveTT.png

    When saved, it will be displayed in the list of existing Term Type templates.

Edit a Custom Template
-------------------------

1. Log in to Alation as a Catalog Admin or Server Admin.
2. If you are a Catalog Admin, mouse over the settings icon on the right of the main toolbar:
       
    .. image:: ../../_static/Screen_Shot_2018-10-16_at_2.17.18_PM.png

    In the menu that opens, click **Customize Catalog**. The **Customize Catalog** page will open:

    .. image:: ../../_static/Customize_catAdminPath.png

    .. note::

         If you log in as a Server Admin, go to **Admin Settings** > **Catalog Admin** section > **Customize Catalog** to open the **Customize Catalog** page.

3. Click the **Custom Templates** tab.
4. Scroll to **Term Types**.
5. Find and click the custom template you want to edit. The editor will open on the right. You can edit the title and add and remove custom fields.
6. Save the changes to the template by clicking the save icon:

    .. image:: ../../_static/Custom_SaveTT.png

Delete a Custom Template
--------------------------

1. Log in to Alation as a Catalog Admin or Server Admin.
2. If you are a Catalog Admin, mouse over the settings icon on the right of the main toolbar:
       
    .. image:: ../../_static/Screen_Shot_2018-10-16_at_2.17.18_PM.png

    In the menu that opens, click **Customize Catalog**. The **Customize Catalog** page will open:

    .. image:: ../../_static/Customize_catAdminPath.png

    .. note::

         If you log in as a Server Admin, go to **Admin Settings** > **Catalog Admin** section > **Customize Catalog** to open the **Customize Catalog** page.

3. Click the **Custom Templates** tab.
4. Scroll to **Term Types**.
5. Click the custom template you want to delete. The editor will open on the right. 
6. Click the **Delete** button in the lower right of the template editor and confirm deletion:

    .. image:: ../../_static/Custom_DeleteTT.png
