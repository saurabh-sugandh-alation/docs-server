Create and View Catalog Sets
=====================================================================

.. include:: /shared/ProductLabels/CloudAndCustomerManaged_Label.rst

Create a New Catalog Set
--------------------------

You must be a Catalog or Server Admin admin to create, edit, and
delete catalog sets. To create a new set, on the **Catalog Sets** page, click the add icon on the upper right:

.. image:: ../../_static/Add_button.png

The new catalog set editor will open:

.. image:: ../../_static/Sets_03.png

You can create sets of two types: **Conditional** or **Manual**. By
default, the **Conditional** type is pre-selected. To create a manual set,
click the **Manual** tab on the editor.

Refer to the dedicated articles for each type to learn more about their properties:

-  :doc:`Conditional Catalog Sets <../../steward/DataDocumentation/ConditionalCatalogSets>`
-  :doc:`Manual Catalog Sets <../../steward/DataDocumentation/ManualCatalogSets>`

View Catalog Sets
--------------------

You can find all catalog sets on the **Catalog Sets** page of your Alation catalog.

.. note::

     A user with :doc:`any role </welcome/CatalogBasics/RolesOverviewVR7>` in
     Alation can view catalog sets. However, you must be a Catalog or Server Admin admin 
     to create, edit, and delete them.

To open the **Catalog Sets** page:

1. Click the **Curate** menu on the main toolbar then click **Catalog Sets**. The **Curate** menu will be available if the :doc:`Stewardship Dashboard <../../steward/AnalyticsStewardship/StewardshipDashboard>`  is enabled in your instance and if you are wearing the :doc:`curator hat <../../steward/AnalyticsStewardship/StewardshipDashboard>`:

    .. image:: ../../_static/Sets_04.png

2. Click the gear icon on the main toolbar to open the **Admin Settings**:

    .. image:: ../../_static/Screen_Shot_2018-10-16_at_2.17.18_PM.png

    In **Admin Settings**, under **Catalog Admin**, click **Catalog Sets**:

    .. image:: ../../_static/Sets_02.png

All existing sets are listed in **All Catalog Sets** table. You can:

-  Open the properties page of a specific set by clicking its title in
   the **Title** column.
-  See the type of the set in the **Rule Definition** column.
-  Sort sets by **Title**, **Member Count**, **Object Type**,
   or **Created By**. To sort, click the corresponding column title.
-  Change the number of rows per page using the **Show N rows** list:

.. image:: ../../_static/Sets_01.png
