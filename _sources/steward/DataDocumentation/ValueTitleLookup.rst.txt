Value Title Lookup
=====================================================================

.. include:: /shared/ProductLabels/CloudAndCustomerManaged_Label.rst

Value title lookup lets a Server Admin connect values in a column to their logical names in a lookup table.

Use these steps to apply value title Lookup: 

#. Go to the catalog page of the column object that needs titles.
#. Under the **Column Sample** section, you will see a blue box with the text **Automatic Titling: Are the titles for these values listed somewhere in the data source?**

    .. image:: ../../_static/ValueTitleLookup_BlueBox.png
        :width: 400px
        :class: with-border

3. Click the **Yes** button.
4. Add the name of the relevant lookup table.

    .. image:: ../../_static/auto1.png

4. Click **Confirm**.

    .. image:: ../../_static/auto2.png

5. Refresh the page.
