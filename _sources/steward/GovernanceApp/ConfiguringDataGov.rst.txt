Data Governance Page
--------------------

.. include:: /shared/ProductLabels/CloudAndCustomerManaged_Label.rst

*Applies from version 2021.4*

Alationâ€™s Data Governance Page allows for access to the different aspects of the Data Governance App such as the Governance Dashboard, Workflow Center, and the Policy Center. Note that this app is priced and packaged separately from the Alation Data Catalog. Contact your account manager to learn more about pricing and licensing before enabling.

Enabling and disabling the Data Governance Page in Alation requires the **Server Admin** role.

.. image:: ../../_static/sw.png

Enable Data Governance Page
###########################

To enable the Data Governance Page:

1. Go to **Admin Settings** > **Server Admin** > **Feature Configuration**.

.. image:: ../../_static/dg_admin.png

2. Toggle **Alation Data Governance App** to the **On** position.

.. image:: ../../_static/dg_admin2.png

3. Click the **Save changes** button.

.. image:: ../../_static/dg_admin3.png

4. Users may now access the page by selecting the **Apps** menu > **Govern**.

.. image:: ../../_static/sw.png

Disable Data Governance Page
############################

To disable the Data Governance Page:

1. Go to **Admin Settings** > **Server Admin** > **Feature Configuration**.

.. image:: ../../_static/dg_admin.png

2. Toggle **Alation Data Governance App** to the **Off** position.

.. image:: ../../_static/dg_admin4.png

3. Click the **Save changes** button.

.. image:: ../../_static/dg_admin3.png
