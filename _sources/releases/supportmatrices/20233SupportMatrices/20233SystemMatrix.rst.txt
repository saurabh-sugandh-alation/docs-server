Supported Docker Versions
----------------------------

- Docker 18.09.1 or later
- Docker Compose 1.27.0 or later

    .. note::

        From release 2021.2, Docker Engine can be installed using Alation Container Service: :doc:`Install Docker Using Alation Container Service </installconfig/AlationContainerService/InstallDockerUsingAlationContainerService>`
