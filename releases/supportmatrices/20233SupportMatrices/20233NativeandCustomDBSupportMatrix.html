2023.3 Native and Custom DB Connectors¶

Refer to the tables below for information on supported databases, file systems, BI tools, and their versions.

Important

Alation has certified support for the versions in this matrix with release 2023.3. If your Alation instance uses a different release, refer to the Support Matrix corresponding to this release.

If you do not find the version of interest in this Support Matrix, contact Alation Support for clarifications.
